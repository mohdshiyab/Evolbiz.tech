# Evolbiz Technologies Website

A production-ready corporate website developed for **Evolbiz Technologies**, a UAE-based digital solutions company. The website showcases the company's services, provides a modern responsive user experience, and includes a fully functional contact form integrated with MongoDB Atlas.

## 🌐 Live Demo

https://evolbiz-tech-osw6.vercel.app/

## About the Project

This project was developed as a real-world client website for Evolbiz Technologies. It focuses on delivering a professional online presence with a responsive design, fast performance, and secure backend integration for customer inquiries.

## Features

- Responsive modern UI
- Next.js App Router
- Contact Form with backend API
- MongoDB Atlas integration
- Mongoose database models
- TypeScript
- Tailwind CSS
- SEO-friendly structure
- Mobile-first design
- Deployed on Vercel

## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- MongoDB Atlas
- Mongoose
- Vercel

## Environment Variables

Create a `.env.local` file:

```env
MONGODB_URI=your_connection_string
MONGODB_DB=your_database_name
```

## Installation

```bash
git clone https://github.com/mohdshiyab/evolbiz-website.git
cd evolbiz-website
npm install
npm run dev
```
